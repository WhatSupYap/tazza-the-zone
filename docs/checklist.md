## 1. 게임 규칙 및 로직 명확화

### 베팅 시스템 세부 규칙
- [✅] **하프(Half) 베팅 정확한 계산법**
  - 현재: "현재 판돈의 절반"
  - 명확히 필요: 판돈 총액의 절반인지, 상대가 건 금액의 절반인지?
  - 소수점 처리 방법 (올림/내림/반올림)
  
- [✅] **베팅 순서 및 턴 관리**
  - 1차 베팅: 선이 먼저 베팅하는지?
  - 2차 베팅: 순서가 바뀌는지?
  - 베팅 라운드 최대 턴 수 제한 (무한 레이즈 방지)
  
- [✅] **올인 규칙**
  - 보유 금액 부족 시 자동 올인 처리
  - 양쪽 금액이 다를 때 차액 처리 방법
  - 올인 후 즉시 쇼다운인지, 상대가 선택하는지?
  
- [✅] **최대 베팅 제한**
  - 무제한인지, 아니면 판돈의 N배까지인지?
  - NPC 배짱 수치에 따른 베팅 상한선 구체적 수치
  
- [✅] **체크 규칙**
  - 양쪽 모두 체크 시 다음 단계로 진행
  - 상대가 베팅한 후에는 체크 불가능한지?

### 특수 족보 발동 규칙
- [ ] **땡잡이 (3월+7월)**
  - 정확한 승리 대상: 1땡~9땡 모두 포함
  - 광땡은 제외 (1광땡, 3광땡, 삼팔광땡 모두)
  - 땡잡이 vs 땡잡이 시 처리 방법
  
- [ ] **구사 (4월+9월) 재경기 상세 절차**
  - "현재 베팅된 모든 금액을 묻는다" → 판돈을 2배로 만드는가?
  - 재경기 카드 배분: 새로 2장 받는지, 기존 3장 중 2장 선택인지?
  - 재경기 베팅 규칙: 일반 베팅과 동일한지?
  - 재경기에서 또 구사가 나오면?
  
- [ ] **멍텅구리 구사 (4열끗+9열끗)**
  - 구사와의 정확한 차이점
  - "9땡까지 잡을 수 있음" → 9땡에게 이긴다는 의미인지?
  - 다른 땡(1~8땡)도 이기는지?
  
- [ ] **암행어사 (4열끗+7열끗)**
  - 1광땡(1월광+8월광), 3광땡(3월광+8월광) 승리
  - 삼팔광땡(3월광+8월광) 제외 → 3광땡과 삼팔광땡이 다른 족보인지 확인 필요
  - 다른 광땡(예: 1+3광땡)에 대한 처리

### 게임 진행 엣지 케이스
- [ ] 둘 다 1,000원 미만 보유 시 처리
- [ ] 한쪽이 파산 직전(베팅 불가능) 상태
- [ ] 판돈이 0원일 때 (양쪽 체크)
- [ ] 베팅 시간(10초) 초과 시 자동 처리 (자동 다이? 자동 체크?)
- [ ] 무승부 조건 (같은 족보일 때)
